Complexity

Simple exercices

Compute the complexity of the following functions

def sequence(n):
    n = n + 1
    n = n + 2
    n = n * 6
    n = n ** 10
    n = n // 2
    return n

Solution: O(1)

def loop1(n):
    n = n + 1
    i = 0
    while i < n:
        i = i + 1
    n = n * n
    return n

Solution: O(n)

def loop2(n):
    n = n + 1
    i = 1
    while i < n:
        i = 2 * i
    n = n * n
    return n

Solution: O(log n)

def b(t):
    n = len(t)
    for j in range(4):
        for i in range(n):
            t[i] *= t[i]

Solution: O(n)

def h(t):
    n = len(t)
    for i in range(n):
        t[i] = 0
        j = n
        while j > 1:
            t[i] += i*j
            j //= 2

Solution: O(n log n)

Master theorem

def foo(n):
    if n <= 1:
        return
    doOhOne() # doOhOne() runs in O(1) time
    foo(n/2)
    foo(n/2)

Solution: a = 2, b = 2, d = 0, bd = 1 < a, case 3, $O(n^{\log_2 (2)}) = O(n)$

def foo2(n):
    if n <= 1:
        return
    doOhN() # doOhN() runs in O(n) time
    foo(n/2)
    foo(n/2)

Solution: a = 2,b = 2,d = 1, bd = 2 = a, case 1, $O(n \log n)$
